### Week 2 Objectives:
* Deploy compute applications on virtual server environments.
* Manage and manipulate infrastructure using the command-line interface (AWS CLI).
* Grant secure communication permissions between services and host static websites.

### Week 2 Achievements:
* Successfully initialized and accessed an Amazon EC2 instance running Ubuntu/Windows operating system.
* Proficiently configured and invoked service commands using AWS CLI directly within the EC2 instance.
* Understood and successfully configured an IAM Role, allowing EC2 to interact with other services (like S3) without storing hard-coded Access Keys.
* Successfully deployed the Static Website Hosting feature on Amazon S3.
