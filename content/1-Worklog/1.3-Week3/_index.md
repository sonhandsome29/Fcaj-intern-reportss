### Week 3 Objectives:
* Design and build an independent virtual network infrastructure.
* Deploy a fully managed relational database system.
* Integrate and resolve hybrid domain names between local networks and the cloud.

### Week 3 Achievements:
* Fully deployed a secure Amazon VPC network infrastructure with clear Public and Private zones.
* Successfully initialized and configured an Amazon RDS database, ensuring the system is not exposed to the internet.
