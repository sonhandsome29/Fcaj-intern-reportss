### Week 1 Objectives:
* Master the steps of creating an account and managing costs on AWS.
* Get familiar with cloud support tools and development environments.
* Establish basic security and manage access with AWS IAM.

### Week 1 Achievements:
* Successfully created an AWS Account and set up safe cost alerts with AWS Budgets.
* Understood how to contact AWS Support when facing technical issues.
* Safely delegated permissions with AWS IAM, applying the principle of Least Privilege.
* Able to use AWS Cloud9 as a direct code development environment in the browser.
